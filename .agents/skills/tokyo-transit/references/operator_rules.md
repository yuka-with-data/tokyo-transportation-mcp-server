# Operator Rules
## Purpose
This reference defines operator-specific routing behavior and transfer considerations across the Tokyo metropolitan rail network.

Load this reference when:
- routing croses multiple operators
- through-service behavior affects routing
- operator-specific transfer behavior matters
- multiple valid routes require operator-aware prioritization

## JR Lines
- Prefer JR routes for major cross-city travel when transfer count is reasonable
- Use Yamanote Line as a major interchange reference
- Clearly identify rapid, express, and local service differences when relevant

## Tokyo Metro
- Prefer Metro routes for dense central Tokyo travel
- Clearly identify through-service operations when applicable
- Some Metro lines continue onto private railway networks

## Toei Subway
- Treat Toei Subway as interoperable with Tokyo Metro for urban routing
- Identify transfers clearly when switching operators

## Private Railways
Examples include:
- Tokyu
- Keio
- Odakyu
- Seibu
- Tobu
- Keisei
- Keikyu

Routing considerations:
- Some services continue directly into subway networs
- Express service patterns vary by operator
- Transfer requirements may differ by service type
