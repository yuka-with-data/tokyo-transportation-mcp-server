## Core MCP application package for the Tokyo MCP Server
This directory contains the internal architecture, tool definitions, services, and supporting utilities used by the MCP server.

### Directory Purpose
The package is organized around a layered MCP architecture:
- `tools/`
- `services/`
- `services/experimental/`
- `data/`
- `utils/`

> It is not yet intended to be a production-grade transportation platform.
